
## Memory Requirements

					(bytes)
* 3 SSIDs, 3 PSSWDs		:	192
* Device Name			:	20
* UUID				: 	32
* Private Key			:	128
* Song Memory			: 	200
	* 8 beats per quantum
	* 4 Bytes per delay (max. 10ms * 1024)
	* 3 variations on an average per song

Max. EEPROM in ATmega328p = 1kB
Max. SRAM in ATmega328p = 2kB
